Element with Class Selector

CSS : programmeertaal dat kan interageren met de basiselementen gevormd door HTML code om deze elementen dieper te kunnen verwerken (bv. elementen dynamischer maken, meer vorm geven aan de elementen, formateren, lay-out,..) m.a.w. CSS kan de stijl veranderen van de structuren van een website.

Note : CSS en HTML zijn technisch gezien geen programmeertalen, maar talen waarmee respectievelijk (web)pagina structuren-en stijlinformatie kunnen gedefinieerd worden. [1]
